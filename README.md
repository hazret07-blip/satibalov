Описание приложения «Random Quote Generator»
Автор: Сатибалов Артур

Название: Random Quote Generator (Генератор случайных цитат)

Цель проекта: создать удобное GUI‑приложение для генерации случайных цитат с сохранением истории просмотров и поддержкой версионного контроля через Git.

Функциональность
Приложение позволяет:

Генерировать случайные цитаты из предустановленной базы или пользовательской коллекции.

Отображать цитату с указанием:

текста цитаты;

автора;

темы/категории (опционально).

Вести историю просмотренных цитат (последние 10–20 цитат).

Фильтровать цитаты по:

автору;

теме/категории.

Добавлять новые цитаты в базу данных (опционально).

Сохранять и загружать данные в формате JSON:

автоматическое создание файла quotes_data.json при первом сохранении;

сохранение текущей базы цитат и истории просмотров;

загрузка данных при запуске приложения.

Управлять историей:

очистка истории просмотров;

экспорт истории в текстовый файл.

Интегрироваться с Git для версионного контроля:

инициализация локального репозитория;

настройка .gitignore для исключения временных файлов;

коммиты при сохранении данных (опционально).

Технические требования
Язык программирования: Python 3.8+

GUI‑библиотека: tkinter (встроенная)

Хранение данных: JSON (json модуль Python)

Генерация случайных значений: модуль random

Система контроля версий: Git

Зависимости: отсутствуют (для базового варианта)

Пошаговая инструкция по реализации
Шаг 1. Создание GUI‑интерфейса

Используем tkinter для создания окна с:

областью отображения цитаты (текст, автор, тема);

кнопкой «Новая цитата»;

выпадающими списками для фильтрации (автор, тема);

областью истории просмотров (последние цитаты);

кнопками управления («Очистить историю», «Сохранить данные»).

Шаг 2. Реализация генерации случайных цитат

Создаём список предустановленных цитат в виде словарей:
self.quotes = [
    {"text": "Знание — сила", "author": "Фрэнсис Бэкон", "topic": "Мудрость"},
    {"text": "Быть или не быть — вот в чём вопрос", "author": "Уильям Шекспир", "topic": "Философия"}
]
Реализуем метод generate_quote(), который:

выбирает случайную цитату из списка с помощью random.choice();

добавляет выбранную цитату в историю (self.history);

отображает цитату в интерфейсе.

Шаг 3. Сохранение и загрузка данных

Реализуем методы:

save_data() — сохраняет self.quotes и self.history в quotes_data.json;

load_data() — загружает данные из quotes_data.json при запуске (если файл существует).

Шаг 4. Фильтрация цитат

Добавляем функционал фильтрации по автору и теме:

заполняем выпадающие списки уникальными значениями из базы цитат;

реализуем метод apply_filters(), который фильтрует цитаты перед генерацией.

Шаг 5. Управление историей

Реализуем:

отображение последних цитат в текстовом поле или таблице;

метод clear_history() для очистки истории;

возможность экспорта истории в .txt файл.

Шаг 6. Настройка Git

Инициализируем репозиторий: git init.

Создаём .gitignore со следующим содержимым:
__pycache__/
*.pyc
.vscode/
Добавляем файлы и делаем первый коммит:

bash
git add .
git commit -m "Initial commit: Random Quote Generator v1.0"
Создаём репозиторий на GitHub и отправляем код:

bash
git remote add origin <URL-вашего-репозитория>
git push -u origin main
Инструкция по запуску
Установите Python 3.8 или новее.

Клонируйте репозиторий:

bash
git clone [URL-репозитория]
Перейдите в папку проекта:

bash
cd Random-Quote-Generator
Запустите приложение:

bash
python main.py
Примеры использования
Сценарий 1. Генерация случайной цитаты

Запустите приложение.

Нажмите кнопку «Новая цитата».

В интерфейсе отобразится случайная цитата с автором и темой.

Сценарий 2. Фильтрация по автору

Выберите автора в выпадающем списке фильтров.

Нажмите «Применить фильтр».

Следующая сгенерированная цитата будет от выбранного автора (если есть в базе).

Сценарий 3. Сохранение данных

Нажмите кнопку «Сохранить данные».

Проверьте файл quotes_data.json — в нём сохранятся все цитаты и история.

Сценарий 4. Загрузка данных

Закройте и перезапустите приложение.

Данные автоматически загрузятся из quotes_data.json.

История просмотров будет восстановлена.
